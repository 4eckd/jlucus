## Usage Notes

- These ASCII art samples use terminal-safe characters
- Compatible with monospace fonts (JetBrains Mono, Fira Code, etc.)
- Use sparingly for visual impact
- Consider adding neon glow effects via CSS text-shadow
- Ideal for:
  - Loading screens
  - Section headers
  - Terminal UI elements
  - Error/success messages
  - Command palette decorations

## Implementation Example

```tsx
// In a component
const HERO_BANNER = `
     _ _                     ____
    (_) |_   _  ___ _   _ ___(___ \\
    | | | | | |/ __| | | / __|__) |
    | | | |_| | (__| |_| \\__ \\/ __/
   _/ |_|\\__,_|\\___|\\__,_|___/_____|
  |__/
`;

<pre className="font-mono text-primary neon-glow">
  {HERO_BANNER}
</pre>
```
